Creational patterns
- Builder patterns : 
  - Used when a an object with lot of parameters has to be configured
  - used constructors combinations can get explosive here.
  - here for a product, a builder is implemented by custom builders and a master/director kinds of guides how to create a custom builder object
  - https://www.geeksforgeeks.org/builder-design-pattern/

- Factory pattern
  - Dependency upon the input/request type, we have got to create different implementation of the same interface.
  - example: https://www.geeksforgeeks.org/factory-method-for-designing-pattern/
